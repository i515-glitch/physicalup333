**안티그래비티 개발팀 전달용 · 로컬 데스크톱 앱 구축 지시서**

## 0. 한 줄 요약
온라인 설문 + 인바디 사진 업로드를 받아, **맥북 로컬에서** BioCode를 계산하고 프리미엄 PDF 리포트를 생성하는 **단독 실행형 데스크톱 앱**. 외부 클라우드 API·서버 없이 100% 오프라인 구동.

## 1. 전체 데이터 흐름
```
[사용자 입력]
  ├─ 설문 응답 (정신·위장·수면·활동)
  ├─ 키 / 몸무게 / 생년월일시
  └─ 인바디 사진 (유료) ── 업로드
         ↓
[로컬 Gemma 비전 OCR]  ※ 8-6
  사진 → 신장·체중·골격근량·체지방률 추출
         ↓
[엔진 연산]  P333 엔진 .xlsx 매트릭스 로직
  7입력 → 선천40·대사35·생활25 → 3축 점수 → 단계 → BioCode(3자리)
         ↓
[27패턴 매칭]  코드 → 캐릭터명 + 출현율 + 처방
         ↓
[파이썬 차트 렌더]  ※ 8-7  Matplotlib/Seaborn
  오행 밸런스 차트 + 목표 스펙 비교 차트
         ↓
[PDF 리포트 생성]  보고서 양식에 주입 → 고객 전달
```

## 2. 핵심 모듈 명세

### 2-1. 인바디 사진 판독 (로컬 Gemma OCR)
- **목적**: 사용자가 골격근량(SMM)·체지방률(PBF)을 직접 타이핑할 때 생기는 이탈·오타 제거
- **방식**: 외부 유료 클라우드 API 호출 금지. 맥북에 상주하는 구글 오픈소스 LLM **Gemma**를 로컬 포트로 연동
- **입력**: 인바디 용지/화면 사진 (jpg/png)
- **처리**: 비전 프롬프트로 사진 판독(OCR) → 신장·체중·골격근량·체지방률만 정량 추출
- **출력**: 추출 수치를 엔진 입력 레이어에 자동 주입 (사람 타이핑 없음)
- **제약**: 클라우드 통신 없음 → 오프라인 동작, 변동비 0

### 2-2. 엔진 연산 (BioCode 계산)
- **소스**: P333 엔진 .xlsx의 [가중치 매트릭스] 시트 로직을 코드로 이식
- **입력 7종**: 유전키(부모키)·음양오행(만세력)·사상의학체질·심성정 / 인바디·위장설문·수면활동설문
- **축 가중치**: 선천 40% · 대사 35% · 생활 25% (합 100)
- **3축 배분 매트릭스**:
  - 선천: 유전키30 + 음양오행25 + 사상의학25 + 심성정20
  - 대사: 인바디50 + 위장설문50
  - 생활: 수면설문70 + 위장설문10 + 인바디20
- **단계 컷오프 (축별 만점 다름)**:
  - 선천(40점): 0~14=1 / 14~28=2 / 28~40=3
  - 대사(35점): 0~11=1 / 11~24=2 / 24~35=3
  - 생활(25점): 0~8=1 / 8~17=2 / 17~25=3
- **BioCode**: 선천-대사-생활 단계를 이어붙인 3자리 (예: 1-2-2)
- **만세력**: 생년월일시→오행 변환은 별도 만세력_DB.xlsx(3289행) 참조. AI API 없이 룰베이스 오프라인 연산
- **종목 보정(1.15)**: 목표 스펙 표시에만 적용. BioCode·예상키 등 기준 숫자엔 **절대 미적용**

### 2-3. 차트 렌더 (파이썬 그래픽)
- **원칙**: LLM 이미지 생성 사용 금지. Matplotlib/Seaborn으로 수식 기반 렌더 (같은 입력=같은 그림 보장)
- **차트 1 — 오행 밸런스**: 만세력 오행(목화토금수) 분포를 막대/방사형으로. 과다·결핍 시각화
- **차트 2 — 목표 스펙 비교**: 현재 인바디 수치 옆에 목표 레벨(KBO·대학) 커트라인 바를 나란히

### 2-4. 패키징 (오프라인 로컬 앱)
- **형태**: 단독 실행형 데스크톱 앱 (Mac/Windows). 안티그래비티 로우코드로 외형, 연산은 로컬 하드웨어(M칩)
- **인프라비**: 외부 서버(AWS)·AI API 없음 → 고정비 0
- **보안**: 아동 신체 데이터 외부 전송 없음 → 개인정보 위험 최소화. 오프라인(야구장·전지훈련장) 구동

## 3. 무료 / 유료 구분
| 구분 | 입력 | 출력 |
|------|------|------|
| 무료 | 설문 + 키·몸무게·생년월일시 (BMI까지) | BioCode 코드 + 캐릭터 + 방향 힌트 |
| 유료 | + 인바디 + 부모키 + 목표 + 추가문항 | 정밀 분석 + 상세 운동 스케줄 + 로드맵 |

※ **코드는 무료·유료가 같아야 함** (인바디 자리를 무료는 키·몸무게·BMI로 대체, 비중 25% 고정 → 코드 안 흔들림)
