# ⚙️ PHYSICAL UP 분석엔진 — 역방향 설계서

> ★ VERSION: v1 (2026-06-23)
> 원칙: "결과부터 거꾸로" — 최종결과 확정 → 필요점수 → 필요문항
> 흐름: 타고난 것 → 현재 것 → 미래 것
> 메시지: "타고났으나, 더 클 수 있다"

═══════════════════════════════════════
## 0. 엔진 한눈에 (입력→처리→출력)
═══════════════════════════════════════
```
[입력 4개]
 ① 부모키(아빠·엄마)  ② 생년월일·생시  ③ 설문  ④ 인바디
        ↓
[처리 — 타고난>현재>미래]
 A. 타고난 것:  부모키→예상키 / 생년월일→체질경향(3축 바탕)
 B. 현재 것:    설문→생활·심리점수 / 인바디→발달점수
 C. 격차 추적:  예상 vs 현재 → 원인 범인찾기
 D. 종합:       BioCode 3축 + 27유형
 E. 미래 것:    목표격차 → 시기별 로드맵·처방
        ↓
[출력 = 보고서 채움]
 예상키 / 6영역판정 / 백분위 / BioCode / 인과1문단 / 로드맵 / 처방
```

## 2. [타고난 것] — A모듈
═══════════════════════════════════════
### A-1. 부모키 → 예상키 (O1)
- 입력: 아빠키, 엄마키, 아이 성별
- 공식(MPH): 남아=(부+모+13)/2, 여아=(부+모-13)/2  (±5cm 오차)
- 출력: "유전적으로 OOcm 전후 기준선"
- ★연령별 현재 예상치도: 예상 성인키 → 현재 나이 백분위로 환산
  (예: 성인예상 175 → 14세 시점 기준선 ≈ 150)  ※성장도표 곡선 필요

### A-2. 생년월일 → 체질 경향 (O2) — 간이판정·참고
- 입력: 생년월일·생시
- 출력: 3축 바탕 경향(참고)
  - 몸 바탕: 대사·소화 경향 (저장형↔소모형)
  - 활동 바탕: 활동 성향 (정적↔동적)
  - 마음 바탕: 기질 (예민↔느긋)
- ★간이판정: 복잡한 명리X. "타고난 경향 힌트"만. "~경향" 톤.

## 4. ★격차 추적 — C모듈 (엔진의 심장)
═══════════════════════════════════════
```
1. 예상치(A-1) vs 현재(B-2) → 격차 계산
   예: 14세 예상 150 vs 현재 145 → -5cm ⚠️
2. 격차 있으면 → 원인 범인찾기 (B-1 설문에서)
   - 수면 점수 낮나? → 수면이 원인
   - 식습관 점수 낮나? → 식사량이 원인
   - 회복 낮나? / 흡수(체질) 문제? 
3. 출력: "예상보다 OO 뒤처짐. 원인은 OO로 보임. 이대로면 잠재력 못 채움."
```
- ★우선순위: 격차 큰 영역 = 최우선 처방 대상
- 안전: "~로 보인다" 톤 / "이대로 가면" = 경고지 협박 X / 의학신호는 전문의

마음 축 = 체질 기질바탕(A-2) + 설문 기질·스트레스(B-1)→ 1~3단계
        ↓
BioCode 3자리 (예 131) + 27유형 캐릭터명
