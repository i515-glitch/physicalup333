# 🧭 PHYSICAL UP 보고서 — 핵심 논리 골격

> ★ VERSION: v1 (2026-06-23)

═══════════════════════════════════════
## 한 줄 논리 (4단계)
═══════════════════════════════════════
현재상태 완벽분석 → BioCode로 정의 → 목표 설정 → 나아갈 길 제시

```
1. 현재상태 완벽분석   나를 안다 (입력 3개 종합)
2. BioCode로 정의      나를 한 코드로 (3입력의 결합 결과)
3. 목표 설정           어디로 갈지 안다 (선수 기준)
4. 나아갈 길 제시       어떻게 갈지 안다 (자기발전 로드맵)
```
→ 무료333은 "1·2의 간이판". 유료는 1~4 전부 + 완벽분석.

═══════════════════════════════════════
## 입력값 3개 = 현재상태 완벽분석의 재료
═══════════════════════════════════════
| 입력 | 무엇을 보는가 | 333 축 연결 | 성격 |
|------|--------------|------------|------|
| ① 생년월일·생시 | 타고난 몸 (체질) | [몸] 바탕 | ★타고난 것 |
| ② 설문 | 현재 생활패턴 + 신경성·정신상태 | [활동]+[마음] | 지금 사는 법·마음 |
| ③ 인바디 | 현재 근육·발달 상태 | [몸] 현재값 | 지금 몸 |

★핵심: 한 개만 보면 무료333 수준. **셋을 합쳐야** 아이를 반 이상 맞춘다.
- ①타고난 출발점 + ②지금 사는 법 + ③지금 몸 → "왜 이 상태인지" 설명됨
- ①은 "타고난 것" / ②③은 "만들어진 것" → 둘을 구분해줘야 진짜 분석

(주의: ①은 '사상의학'이라 표면 표기하지 않음. '타고난 체질'로 표현. 
 단정 금지, "~경향이 있다고 알려져 있다" 톤. 의료신호는 전문의.)

═══════════════════════════════════════
## BioCode = 3입력의 결합 결과
═══════════════════════════════════════
```
[①생년월일] → 타고난 체질 점수 (몸의 바탕)
[②설문]     → 생활 점수 + 심리 점수 (활동·마음)
[③인바디]   → 현재 발달 점수 (몸의 현재)
        ↓ 종합·교차
   BioCode (몸·활동·마음 3축, 각 1~3단계) + 27유형 캐릭터명
```
→ BioCode는 "결과"가 아니라 "현재상태 완벽분석을 한 좌표로 압축한 것".
→ 그래서 다음 단계(목표·로드맵)로 연결되는 다리가 된다.

═══════════════════════════════════════
## 목표 = 기존 선수들의 실측치 (지어낸 값 아님)
═══════════════════════════════════════
"운동 잘하는 몸"의 기준 = 실제로 그 레벨에서 뛰는 선수들.
사다리: 선수 또래 상위10% → 대학 → KBO프로 → NPB → MLB
- 비교 기준도 "일반 또래"가 아니라 "선수 또래 상위권"
- 데이터: 야구목표기준DB 참조

═══════════════════════════════════════
## 나아갈 길 = 목표에 맞춘 자기발전 로드맵 (★돈값)
═══════════════════════════════════════
현재(BioCode) → 목표(선수기준)까지의 격차를 
중·고·대 시기별로 메우는 단계별 경로.
- 이게 보고서의 본체이자 부모가 돈 내는 이유.
- 분석(1·2)은 이 로드맵(4)을 그리기 위한 근거.

## ◆ 입력값 보강 (2026-06-23 추가) — 4입력 + 시간 흐름

### 시간 흐름 척추 (★보고서·엔진의 중심 논리)
타고난 것 → 현재 것 → 미래 것
```
[타고난 것]  부모키(유전) + 생년월일(체질·성향)   "왜 이렇게 태어났나"
[현재 것]    설문(생활·심리) + 인바디(현재 몸)      "지금 어떤가"
[미래 것]    목표(선수기준) + 로드맵                "어디로, 어떻게"
```
- 타고난 것=이해한다 / 현재 것=분석한다 / 미래 것=돕는다 (채널 철학)

### 최종 입력값 4개
| 입력 | 보는 것 | 시간축 | 성격 |
|------|--------|--------|------|
| ① 부모 키(아빠·엄마) | 유전적 신체 잠재력 | 타고난 것 | ★객관(숫자) |
| ② 생년월일·생시 | 체질·성향 | 타고난 것 | 참고(맹신X) |
| ③ 설문 | 생활패턴+신경성·정신 | 현재 것 | 핵심 |
| ④ 인바디 | 현재 근육·발달 | 현재 것 | 핵심 |

### ① 부모 키 → 예상 키 (유전 기준선)
- 중간부모키(MPH) 공식:
  - 남아 예상키 ≈ (아빠키 + 엄마키 + 13) ÷ 2  (±오차)
  - 여아 예상키 ≈ (아빠키 + 엄마키 − 13) ÷ 2  (±오차)
- ★용도: "유전적으로 이 정도 잠재력. 후천 노력으로 더 갈 수 있다" (참고 기준선)
- ★주의: 단정 금지. 부모보다 큰 사례 많음(영양·운동·시대). "기준선일 뿐" 톤.

### ②③④에 체질이 3축 전부로 스며듦 (사상의학)
생년월일(체질) 하나가 몸·활동·마음 3축 바탕에 모두 입력:
- 몸 축 ← 체질의 신체 경향(대사·소화·골격)
- 활동 축 ← 체질의 행동 성향(에너지·활동 패턴)  ← 추가
- 마음 축 ← 체질의 기질·성정(예민·느긋 등)      ← 추가
→ 각 축 = "체질 바탕(참고) + 현재값(설문·인바디)"
→ "타고난 결 + 지금 만들어진 것"이 축마다 겹쳐 보임

### 체질·성향 취급 원칙 (★중요)
- 참고 입력(O) — 타고난 경향 힌트, 3축 시작점
- 단정 판정(X) — "당신은 ○○인이라 ○○하다" 금지
- 최종 판단은 현재 데이터(설문·인바디) 우선. 체질은 거들 뿐.
- "~경향이 있다고 알려져 있다" 톤 / 의료신호는 전문의
- "맹신 아니라 참고" — 타고난 것 + 만들어진 것 둘 다 본다

## ★★ 핵심 메시지 (슬로건) — "타고났으나, 더 클 수 있다" ★★

보고서 전체를 관통하는 한 줄. 유전·체질을 넣되 '운명'이 아닌 '출발점'으로.

```
타고났다 (인정)          더 클 수 있다 (희망)
부모키 = 유전 잠재력  →   후천 노력으로 그 이상
체질 = 타고난 성향        습관·훈련으로 바뀜
"이게 출발점이야"        "출발점이 끝이 아니야"
```

### 시간 흐름에 메시지 입히기
- [타고난 것] "너는 이렇게 태어났어" (부모키·체질) → 인정·이해
- [현재 것]   "지금은 여기 있고" (설문·인바디)     → 분석
- [미래 것]   "그런데 더 클 수 있어" (목표·로드맵)  → ★희망

### 근거 (논문)
- 부모보다 큰 아이 많음(영양·운동·시대) → 유전이 천장 아님
- 중학교때 작다 고교 급성장 사례 → 타고난 게 다 아님
- 체격 작아도 하체·코어로 정상급(알투베·김선빈) → 체격 극복 가능
- 청소년기 = 신경적응 골든타임(2022) → 지금 노력이 진짜 바뀜

### 효과
부모가 '안심(타고난 걸 앎)' + '희망(더 갈 수 있음)'을 동시에 받음.
"작아서 안 되겠네"(X) → "출발점은 이렇지만 이렇게 하면 큰다"(O)

## ★★★ 엔진 핵심 추론 — "격차 추적" 로직 (2026-06-23) ★★★

엔진이 살아 움직이는 방식. 타고난 예상치와 현재값의 '격차'가 분석의 방아쇠.

### 작동 예시
```
타고난 것:  부모키 → 14세에 150cm 예상 (유전 기준선)
현재 것:    실측 145cm
   ↓
격차:       예상 -5cm ⚠️ (뒤처짐 신호)
   ↓
원인 추적:  왜 뒤처졌나? 엔진이 범인 찾기
   - 수면 부족? (설문)
   - 식사량 적음? (설문)
   - 생활습관? (설문)
   - 흡수·대사? (인바디·체질)
   ↓
경고+처방:  "이대로 가면 못 큰다 → (원인)부터 바꿔라"
```

### 4단계 로직
| 단계 | 처리 | 입력 |
|------|------|------|
| 1. 유전 예상치 | 부모키 → 예상키(MPH) | 부모 키 |
| 2. 격차 계산 | 예상 vs 실측 → 차이 | + 인바디 |
| 3. 원인 추적 | 격차의 범인 찾기 | 설문·체질 |
| 4. 경고+처방 | "이대로면 못 큰다" + 해결책 | 종합 |

### 왜 핵심인가
- 예상치(기준선)가 있어야 "뒤처졌다"가 보임 → 분석이 시작됨
- 부모키는 단순 참고 X → **격차를 만드는 기준선** (방아쇠)
- "마르셨네요"(X) → "150 가야 하는데 145, 원인은 수면, 이대로면 못 큼"(O)
- 이게 부모가 돈 내고 듣고 싶은 말 = 돈값

### 단, 안전 원칙
- 예상치는 "기준선일 뿐" — 단정 X, 부모보다 큰 사례 많음
- "이대로 가면" = 가능성 경고, 협박 아님
- 격차 원인은 "~로 보인다" 추정 톤
- 키 안 크는 의학적 신호는 전문의(성장클리닉) 안내
