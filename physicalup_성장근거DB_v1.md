## 1. 성장도표 (예상키·백분위의 뼈대) ★공식

  - 연령별 신장·체중·BMI 백분위 (3~18세)
  - 공공데이터포털에서 **LMS값(L·M·S) 다운로드 가능** → 엔진이 직접 백분위 계산

- ★엔진 활용:
  - 현재 키 → 연령·성별 백분위 산출 (예: 14세 145cm = 하위 OO%)
  - "예상 vs 현재" 격차의 기준선
- 근거: "성장은 여러 요인 복합작용하나 일정 부분 예측 가능한 패턴"(서울대 김재현 교수)

## 2. 유전 vs 환경 (★"타고났으나 더 클 수 있다" 핵심근거)
═══════════════════════════════════════
- 키 결정 = **유전 약 70~80%** (대규모 메타분석) → 부모키가 기준선인 이유
- 단, 영양·수면·운동·질환도 큰 영향:
  - 영양부족·수면장애·만성질환 → 부모 키보다 작아질 수 있음
  - ★균형식사+충분수면+꾸준운동 → **유전적 키 이상으로도 성장 가능**
- 출처: 소아내분비 전문의(하이닥 2025)
- ★이게 우리 슬로건의 의학적 증거: 타고남(유전70~80%) + 더큼(환경으로 그이상)

═══════════════════════════════════════
## 3. 수면 → 성장 (격차 원인: 수면)
═══════════════════════════════════════
- 깊은 잠 시 성장호르몬 **평소보다 4.5배** 분비
- 중고생 대부분 수면 8시간 미만 → 만성 수면부족이 키 성장 막을 수 있음
- 성장기엔 수면 중 성장호르몬 대량 분비(밤 10시~새벽 2시 최다)
- 수면장애 시 성장발육 뒤처짐 "증명됨"
- ★질보다 양 아닌 "질" 중요 — 숙면(빛·소음 차단, 카페인 X)
- 출처: 소아청소년과 전문의(YTN), 대우병원 진단검사의학과
- ★엔진 활용: 설문 수면 점수 낮음 → 격차 원인 후보 1순위

═══════════════════════════════════════
## 4. 저신장 기준 (격차 경고 컷오프)
═══════════════════════════════════════
- 의학적 저신장 = 같은 연령·성별 중 **하위 3백분위 미만 (-2 SD 미만)**
- ★엔진 활용: 현재 키가 하위 3% 미만 → "전문의(성장클리닉) 상담 권고" 신호
- 성장호르몬 치료 시 최종키 평균 4~6cm 증가 연구도 있음(참고)

## 3-B. 수면 → 성장 (상세 보강, 2026-06-23 추가)
═══════════════════════════════════════
### ★국내 논문 — 서파수면 비율 차이 (구체 수치)
- 논문: 특발성 저신장 아동의 야간 성장호르몬 분비와 수면양상 (대한소아과학회지)
- ★저신장 아동은 깊은 수면(서파수면, SWS) 비율이 정상보다 유의미히 낮음:
  **16.4% (저신장) vs 20.8% (정상)**
- 성장호르몬은 잠든 후 1~2시간 깊은 수면 단계에서 가장 왕성히 분비
- → "양(시간)만 아니라 질(깊은 잠)이 중요" / 자주 깨면 분비 감소
- ★엔진 활용: 수면 '질' 문항 필요 (총 시간 + 깊이/중간깸 여부)

### 권장 수면시간 (연령별)
- 유아(3~5세): 10~13시간 / 초등(6~12세): 9~11시간
- 중고생: 대부분 8시간 미만 = 만성 수면부족 (성장 방해)

### 수면 외 발달 영향 (보너스 근거)
- 영유아 길고 규칙적 밤수면 → 인지·언어 발달점수↑ (Bayley-III, KBIT-2)
- 10시간 미만 지속 → 발달지연 확률 1.1~1.15배↑ (일본 대규모 코호트)
- 짧은 수면 → 렙틴↓ 그렐린↑ → 체지방↑ (비만 위험)

### 수면 환경 (처방용)
- 밤 10시 전후 깊은 잠 진입이 핵심
- 블루라이트(스마트폰) 차단 + 어두운 환경
- ★"일찍 많이"보다 "규칙적으로 깊게"

### 출처/반영
- 대한소아과학회지(저신장-수면), Oxford Sleep Journal, Frontiers Public Health, Obesity Journal
- 반영: O (격차추적 수면원인 + 처방). 원문 PDF 확보 권장.

## 6. 스트레스 → 성장 (★[마음] 축 핵심근거, 2026-06-23 추가)
═══════════════════════════════════════
### ★코르티솔 메커니즘 — 마음이 몸 성장을 직접 막는다
- 스트레스 → 코르티솔(스트레스호르몬) 과다분비 → **성장호르몬(GH) 분비 직접 억제**
- 코르티솔이 성장판 세포분열도 방해
- 영양 충분해도 만성스트레스 아동은 연간 수 cm 성장 뒤처질 수 있음(증명)
- ★환경 개선되면 성장속도 급격히 회복 ("정서사회적 소인증" Psychosocial Dwarfism)
- 출처: 소아내분비학 저널 / 대한소아내분비학회지
- ★엔진 핵심: [마음] 축 스트레스 = 격차추적의 직접 원인. "마음→몸" 인과 증명

### 뇌 발달 (보너스)
- 만성 스트레스 → 해마(기억)·전두엽(감정조절) 부피↓, 편도체(불안) 과활성
- → 학습저하·불안·우울 확률 배 이상↑
- 출처: PNAS / 대한신경정신의학회

### 면역 → 성장 에너지 (연결고리)
- 스트레스 → 면역력↓ → 잦은 잔병치레 → 몸이 에너지를 '성장' 아닌 '회복'에 소모
- → 체중·골격 성장 지연
- 출처: Psychosomatic Medicine

### ★축 간 교차 증명 (우리 3축 논리 뒷받침!)
- 스트레스 → 자율신경 자극 → **위장관 운동↓ → 흡수 저하** (마음→몸 흡수)
- 스트레스 → **깊은 잠(서파수면) 방해 → 성장호르몬↓** (마음→활동→몸)
- → "마음이 몸·활동에 다 영향" = 우리 3축 교차 구조의 학술 증거

### 처방 (가장 효과적)
- ★부모와의 안정적 정서 유대감 (← 채널 대원칙 "응원·부모가 먼저")
- 규칙적 신체놀이(운동) → 코르티솔 분해 + 세로토닌 분비 → 성장판 자극
- → "비교·질책"이 코르티솔 올리고, "응원·유대"가 내린다

### 반영
- 반영: O ([마음]축 핵심 + 격차추적 심리원인 + 3축교차 증명 + 처방)
- 원문 PDF 확보 권장 (소아내분비학회지 우선)

## 7. 기질·성격 → 성장 (★[마음] 축 기질 단계 근거, 2026-06-23 추가)
═══════════════════════════════════════
> 핵심: 성격이 키를 직접 바꾸진 않지만, 성격에 따라 스트레스 취약성·수면·식습관·활동량이
> 달라져 결과적으로 성장 차이를 만듦. → 우리 [마음]축 기질 1~3단계의 직접 근거.

### 유형 1) 예민·불안형 (행동억제 기질)
- 같은 환경도 '위협'으로 인식 → 코르티솔 더 자주·많이 분비 → 성장호르몬 방해
- 교감신경 긴장 → 소화기 위축 → 까다로운 식성·흡수율↓ → 성장 더딤
- ★처방: "많이 먹어라/키 커야지" 압박이 오히려 성장 막는 스트레스.
  정서적 안정이 성장호르몬 분비의 지름길.
- 출처: 대한소아청소년과학회(기질-성장), 코르티솔 반응성 연구

### 유형 2) 충동·자극추구형 (외향·활동형, ADHD경향)
- 규칙적 식습관 어려움 + 보상심리로 고열량 과다 → 비만 위험↑
- 활동적 성격은 성장판 자극·골밀도↑ (긍정) / 단 과도한 자극·수면부족은 저해
- ★처방: 규칙적 운동 루틴 + 수면의식(불끄기·독서)으로 진정시키는 환경 통제
- 출처: Obesity Reviews, 스포츠과학·의학 저널

### 유형 3) 안정·회복탄력형 (낙천형)
- 부정적 환경에서도 성장지연(소인증) 확률 현저히 낮음 — 낙천성이 내분비 '완충재'
- 정서안정 → 서파수면 쉽게 진입 → 성장호르몬 왕성
- 출처: Child Development 저널 (회복탄력성)

### ★엔진 활용 (마음 축 3단계 = 기질 유형)
- 우리 [마음] 1~3단계가 이 3유형과 정합:
  - 1단계(민감/불안형) = 유형1 → 정서안정 처방 우선
  - 2단계(안정형) = 유형3 → 완충재, 양호
  - 3단계(활달/강심장형) = 유형2 → 과열·수면 관리 필요
- ★기질은 타고남(체질·생년월일 바탕) + 양육으로 결과 바뀜 = "타고났으나 더 클 수 있다"

### ★부모 양육이 결과를 바꾼다 (채널 핵심)
- "성격은 유전 기질 강하나, 부모 양육방식에 따라 신체에 미치는 호르몬적 결과는 바뀜"
- → 비교·압박(코르티솔↑) vs 안정·유대(성장호르몬↑)
- → 우리 대원칙 "꾸중 No 응원 Yes, 부모가 먼저"의 직접 근거

### 반영
- 반영: O ([마음]축 3단계 근거 + 27유형 기질축 + 양육처방)

## 8. ★2차 성징 & "털 나면 키 멈춘다" 진실 (2026-06-23) — 채널 핵심 떡밥
═══════════════════════════════════════
> 부모 최대 공포: "겨드랑이/음모 털 나면 곧 키 멈춘다" → 절반만 맞음. 순서가 핵심.

### ★"털 나면 끝" = 오해. 진실은 '순서'
- 사춘기 첫 변화 = 음모·겨드랑이 털 나기 "시작" = ★급성장 시작 신호 (한창 클 때!)
- 남자: 고환 커짐(첫신호) → 키 부쩍 → 털 나며 성장 "점차 둔화"
- ★즉 털이 "나기 시작"=급성장중 / 털이 "다 굵어진 후반"=둔화
- 부모는 이걸 뭉뚱그려 "털=끝"으로 오해

### ★사춘기에도 20cm+ 더 큰다 (핵심 수치)
- 남자: 뼈나이 13세 사춘기 시작 → 월 1cm+ 성장
  → 절정기 15세까지 약 16.5cm → 18세까지 +6cm → 사춘기 총 22.5cm+
- 여자: 뼈나이 11세 시작 → 13세까지 약 15cm → 16세까지 +6cm → 총 20cm+
- 여자 초경 후: 약 3~5cm 더 자라고 멈춤 (초경=후반 신호)
- ★"털 보여도 아직 한참 큰다"

### ★2차 성징 시기 (남/여)
| | 사춘기 시작(첫신호) | 2차성징 | 성장 멈춤 |
|--|------------------|---------|----------|
| 남 | 13세경(고환↑) | 13~15세 | 16~18세 |
| 여 | 11세경(유방↑) | 11~13세 | 15~16세 |
- 첫 신호: 남=고환 커짐 / 여=유방 발달 (털·변성기는 그 다음)
- 순서(남): 고환↑ → 키 급성장 → 음모·겨드랑이·턱털 → 변성기 → 둔화

### ★늦게 와도 괜찮다 (만숙형 = 우리 메시지)
- 동년배보다 1~2년 늦는 건 "늦다고 하기도 힘든 수준"
- 변성기·음모가 중1~2에야 시작되는 경우도 정상
- ★만숙형: 남들 멈출 때(16세) 우리 애는 그제야 급성장 → 역전 가능
- "아직 털 안 났다"=걱정 X, 늦게 크는 잠재력일 수 있음

### 성조숙증 경계 (의료 신호)
- 남아 만 9세 이전 고환 커짐 / 여아 만 8세 이전 유방 발달 = 성조숙증
- → 너무 일찍 시작하면 성장판도 빨리 닫혀 최종키 손해 → 전문의
- ★조숙=빨리 크지만 일찍 멈춤 / 만숙=늦게 크지만 오래 큼

### ★엔진·콘텐츠 활용
- 격차추적에 "사춘기 단계" 변수: 털 초기=한창 클 때(낙관) / 후반=마무리
- 입력에 2차성징 체크(고환·음모·변성기·여드름 단계) → 성숙도 간이 추정!
  (병원 골연령 대신 쓸 수 있는 현실적 대안)
- ★유튜브 떡밥: "겨털 났다고 키 끝? 천만에요 — 22cm 더 큽니다"
- 출처: 소아청소년과 Q&A / 성건강정보 / 사춘기 발달 일반. ★원 의학자료 보강 권장

## 9. ★"발 크면 키 큰다" 진실 (2026-06-23) — 속설깨기 시리즈 2

### 통계적 사실 (반은 맞음)
- 성인 발길이-키 상관 0.7+ (높은 양의 상관)
- 성인 발길이 ≈ 키의 약 15%
- 발 큰 무리가 평균 키 큼 = 통계적 사실

### ★사지말단 성장법칙 (발이 '예고편'인 이유)
- 원위부→근위부 성장(Distal-to-Proximal): 급성장기엔 말단(발·손)이 먼저 큼
- → 중고생 키는 그대로인데 발만 270~280mm로 커져 신발 자주 바꿈
- → 말단 성장 끝난 뒤, 다리뼈(대퇴골)·척추가 본격 성장 = 키 급성장
- ★"발이 갑자기 커졌다 = 곧 키도 클 타이밍 신호" (희망 메시지!)

### ★발 크기로 키 보장 안되는 이유 (성장판 폐쇄 순서!)
- 발 성장판: 가장 먼저 닫힘 (만 13~15세)

- → ★최종 키 = 가장 늦게 닫히는 "무릎·척추 성장판" 관리에 달림
- 발 작아도 무릎 성장판 오래 열리면 예상 초월 성장 가능
- 발 커도 무릎·척추 일찍 닫히면(호르몬·영양·스트레스·수면부족) 키 손해

### ★우리 메시지 통합 (모든 근거가 한 곳으로!)

### ★콘텐츠·엔진 활용
- 유튜브 떡밥: "발 크면 키 큰다? 발은 예고편일 뿐, 진짜는 무릎·척추"

- 출처: 법의학 신체계측(발-키 상관) / 사지말단 성장 / 성장판 폐쇄순서. 의학자료 보강 권장

## 11. ★성장판 관리 처방 — 온열·스트레칭·근력운동 (2026-06-23)
═══════════════════════════════════════
> 발·팔다리 자료의 결론("성장판 관리")을 실제 처방으로. 좋은 것 vs 나쁜 것 구분.

### A. 온열 (사우나 vs 찜질) — 구분이 핵심
[좋은 것] 가벼운 온열찜질 (38~40℃, 10~15분, 무릎·종아리)
- 성장판 주변 혈관확장→혈류↑→연골세포 분열 환경 개선
- 성장통 완화→깊은수면(서파) 진입 도움
- ★잠들기 전 시행 + 후 미지근한 물 한 컵(탈수예방)
[나쁜 것] 고온 사우나·찜질방 (50~80℃)
- 아동은 체온조절 미숙→탈수→세포분열 억제
- 고온=위기인식→코르티솔↑→성장호르몬 억제
- ★남아 고환 온도(체온보다 2~3℃ 낮아야)→고온 노출시 발달저해
- ★만 12세 미만 사우나·한증막 금지

### B. 스트레칭 — "키 큰다" 확실한 근거
- 성장판 압박해소(decompression): 낮동안 눌린 성장판→밤에 분열 활발
- ★숨은키 발굴: 굽은등·거북목→디스크 찌그러짐→1~3cm 손실
  스트레칭이 척추 정렬→숨은키 회복 + 성장호르몬 전달효율↑
- 근육연장: 뻣뻣한 근육=뼈 성장 브레이크. 늘려주면 저항↓+성장통↓→숙면
- ★실천: 잠들기 직전 10분(샤워후), 반동X 정적스트레칭 15~30초,
  타겟=햄스트링·기지개·고관절/나비자세

### C. ★근력운동 = 키 안 큰다? = 가장 큰 오해 (우리 채널 핵심!)
- ★"근력운동→성장판 닫힘→키 멈춤" = 대표적 의학 오해(신화)
- 진실: 올바른 근력운동은 키 성장 촉진 + 뼈 튼튼
- 울프의 법칙: 뼈는 물리적 자극에 적응해 더 단단해지고 자람
- 근력운동시 성장호르몬(GH)·IGF-1 더 높은 농도 분비 + 골밀도↑
- ★역도선수 작은건 운동탓 X = 작은체형이 유리해 살아남은 선택편향
- 오해의 진짜원인: 과도한 무게→자세무너짐→성장판 부상→그 부위 조기멈춤
  즉 운동'자체' X, 무리한 무게로 인한 '부상'이 문제
- ★안전 가이드 (미국소아과학회 AAP/NSCA 공식):
  ① 1RM(최대무게) 금지. 12~15회 부드럽게 드는 가벼운무게 2~3세트
  ② 머신보다 맨몸·자유중량 (푸시업·맨몸스쿼트·턱걸이·플랭크·밴드)
  ③ 운동후 단백질·칼슘 + 밤10시전 숙면 세트
- 출처: AAP, NSCA, 울프의 법칙, 스포츠의학

### ★성장판 관리 4대 처방 (최종 정리)
```
1. 수면     밤10시전 깊은잠(서파) — 성장호르몬 4.5배
2. 스트레스  코르티솔 관리 — 안정·유대(부모 응원)
3. 운동     하체자극 + 올바른 근력(맨몸·12~15회) + 스트레칭
4. 자세     척추정렬 — 숨은키 + 호르몬 전달효율
+ 보조: 가벼운 온열찜질(O) / 고온사우나(X) / 영양(단백질·칼슘)
```

## 12. ★점프 운동(농구·배구) = 키 성장 (2026-06-23) — 운동처방 핵심
═══════════════════════════════════════
> "농구·배구하면 키 큰다" = 확실한 사실. 점프=성장판 자극 최고 촉매.

### 1. 지면반발력 = 수직축 자극 (성장판 깨우기)
- 성장판(연골세포)은 수직축 자극에 활발히 분열
- 점프 착지 시 지면반발력 → 발목·무릎·고관절 성장판에 물리적 자극

- ★울프의 법칙(근력운동 근거)과 동일 원리

### 2. 공중 동작 = 천연 스트레칭 (압박 해소)
- 점프 정점 순간 → 중력에서 벗어나 관절·척추 사방 이완 (decompression)

## 15. ★체성분·체중 과학 (살찌우기 = 우리 두번째 기둥) 2026-06-23
═══════════════════════════════════════
> ★채널="마른 운동선수 부모". 키 + 살찌우기(건강한 체중증가)가 두 기둥.
> 마른 선수에게 "어떻게 건강하게 근육·체중을 채우나" = 세계최고 되어야 할 영역.

### A. 칼로리 ↔ 체중 (정량 법칙)
- ★체지방 1kg = 7,700kcal (지방 1g 연소시 7.7kcal)
- 100kcal = 순수 체지방 약 13g
- 매일 100kcal 잉여(증량) → 한달 +390g, 1년 +4.7kg 체지방
- ★역도 증량에도 적용: 마른 선수 체중 늘리려면 일일 잉여칼로리 필요
- 100kcal 체감: 흰쌀밥 3~4숟갈(70g)/우유1컵/사과1개 / 소비=걷기20분·조깅10분

### B. ★체중계 착시 — 글리코겐과 수분
- 글리코겐 1g 저장시 수분 3~4배 끌어당김
- 하루만에 빠지는/찌는 체중 대부분 = '물 무게'(지방 아님)
- 진짜 지방변화는 100kcal당 13g씩 천천히
- ★운동 초기 체중증가 = 지방 아닌 근육내 수분·글리코겐 저장(건강한 신호)
- 코르티솔(스트레스)은 수분 붙잡아 붓게 함 → 체중계 일희일비 금지

### C. ★지방 vs 근육 밀도 (눈바디의 과학)
| 조직 | 밀도 | 1kg당 부피 | 특징 |
|------|------|-----------|------|
| 체지방 | 0.90 g/mL | 약 1,111mL | 가볍고 벙벙 |
| 근육 | 1.06 g/mL | 약 943mL | 단단·꽉참 |
- ★같은 1kg이면 지방이 근육보다 부피 약 1.2배(18%) 큼
  ("지방 부피 3배"=가짜뉴스, 정확히는 1.2배)
- 근육 70~75% 수분 + 고밀도 단백질
- ★지방2kg 빠지고 근육2kg 늘면 체중 동일하나 허리둘레↓ 옷핏↓ (눈바디)
- → ★체중계(무게)보다 인바디(체성분)·줄자가 중요 = 우리 인바디 입력의 근거!

### ★엔진·메시지 활용 (마른 선수 증량)
- 마른 선수 처방: "체중 늘려라"가 아니라 "근육으로 채워라"
  = 잉여칼로리(질좋은) + 근력운동 + 단백질 + 회복
- 격차추적에 체중/체성분 격차 추가: "근육량 또래 하위 → 단백질+근력"
- ★체중계 X 인바디 O 교육 = 부모 불안 해소 (운동초기 체중증가는 건강신호)
- 출처: 영양학 7700kcal 법칙 / 조직 밀도(지방0.90·근육1.06) 생체역학. 의학자료 보강

## 15. ★체중·영양 메커니즘 (살찌우기 축 — 2026-06-23) — 세계최고 목표영역

### A. ★마른 아이 건강한 증량 (살찌우기 해결편)
- ★핵심: 식사량만 늘리면 지방만 늘어 건강 해침 → 근육으로 찌워야
- 증량 단백질: 체중 1kg당 1.2~1.5g (60kg=72~90g/일)
- 하루 5~6끼 조금씩 자주 (마른사람 신진대사 빨라 짧은간격 열량공급)
- ★증량 속도: 하루 +500kcal (급격한 증량 금지, 칼로리 7700=1kg와 연결)
- 단백질 쉐이크 고를때: 단백질 15g+, 당 6g- / 흰우유에 타서
- 운동: 근매스 키우는 6~12회 약간 무거운 무게, 주3회. 큰근육(스쿼트·벤치)
- ★운동 전: 탄수(고구마·바나나) 보충 (공복운동=근손실)
  운동 직후: 빠른탄수+1시간내 단백질 (근생성·증량)
- ★주의: 많이 먹어도 안 찌면 갑상선·장 건강 의심 (검사 권장)
- 출처: 하이닥 영양상담, 살찌우기 가이드

### B. ★장 건강 = 흡수의 뿌리 ("장을 먼저 살리자")
- ★마른 아이 핵심: 장이 망가지면 영양 흡수 안됨 → 먹어도 안 찜/안 큼
- 청소년 장 문제: 오래앉음(장운동↓)+가공식품 선호+학업스트레스 → 변비·흡수저하
- ★장-뇌 축(Gut-Brain Axis): 뇌와 장이 호르몬·면역·신경 신호 공유
  → 유산균 먹으면 스트레스호르몬(코르티솔)↓ 불안↓ (스트레스 근거와 연결!)
- 유산균: 장내 유익균↑→면역력↑+여드름(사춘기) 개선
- ★최소 12주 규칙 복용 (장내 생태계 변화 시간 필요)
- 프리바이오틱스(유산균 먹이) 함께
- 항생제 복용시 장내균 감소→유산균 회복 도움
- ★주의: 가루 유산균 이산화규소 첨가물 (영유아 장면역 미성숙시 주의)
- 출처: 하이닥, 소비자원, 장-뇌축 연구

### C. ★나트륨(짠 거) → 칼슘 빼냄 (탄산 인산과 한 쌍)
- ★짠 거 많이 먹으면: 신장서 나트륨 배출시 칼슘 함께 배출
  → 부족분 채우려 뼈 속 칼슘 빼냄 → 골감소·성장 타격
- 패스트푸드·과자·해산물·내장류 염분 높음 주의
- 칼륨(바나나·감자·키위)이 나트륨 배출 도움
- ★청소년 칼슘 권장: 하루 1300mg (우유 4.5잔). 사춘기 = 뼈 칼슘축적 최대시기
- 비타민D가 칼슘 흡수 도움 (한번에 칼슘 500mg 이하 흡수효율↑)
- 출처: 국민건강보험공단, 대한소아과학회, 서울대 건강지식센터

### D. ★엽떡(엽기떡볶이) = 성장 3중 타격 (우리 콘텐츠 황금소재!)
- ① 매운맛(캡사이신): 위 점막 자극→위장 부담→소화·흡수 저하, 어린 장에 자극
- ② 짠맛(나트륨): 뼈 칼슘 빼냄 (위 C)
- ③ 밀가루 떡(정제탄수): 혈당스파이크→인슐린→성장호르몬 차단 (탄산과 동일기전)
- + 자극적 단짠맵 → 미뢰 마비·도파민 둔화 → 밥 거부 (과자와 동일)
- ★결론: 엽떡 한 그릇 = 흡수저하+칼슘손실+성장호르몬차단+입맛파괴 동시
- ★콘텐츠: "마른 우리 애가 엽떡을 끊어야 하는 과학적 이유"
- (단정 아닌 '자주·과다 섭취' 경계. 가끔은 괜찮다는 균형 톤)

### ★수분 (세포분열의 바탕)
- 물 부족시 세포분열 억제 (탈수→성장판 세포활동 저하, 사우나 근거와 연결)
- 충분한 수분 = 영양 운반·노폐물 배출·세포 활동의 기본
- 탄산·주스 대신 물. 운동 전후 수분 보충

### ★살찌우기 축 완성 (통합)
```
망가뜨리는 것: 과자·탄산·엽떡(입맛·칼슘·성장호르몬·흡수 파괴)
   ↓ 끊고
살리는 순서: ①장 살리기(유산균·12주) → ②흡수력(스트레스·수면)

```

## 18. ★운동 보충식 — 타이밍이 전부 (2026-06-23)

> 포도당캔디·게토레이·단백질바·에너지젤리 = 운동 중·전후 OK / 평소 독
> ★핵심: 같은 당분도 "언제 먹느냐"가 가른다 (우리 혈당스파이크 근거 응용)

### ★대원칙: 혈당스파이크 = 평소엔 독, 운동중엔 약
- 평소: 정제당→혈당급등→인슐린폭발→성장호르몬 차단 (16섹션 근거)
- ★운동 중: 근육이 당을 즉시 소비 → 인슐린 폭발 없이 에너지로 직행
- → 같은 설탕이라도 타이밍이 독/약을 가름

### 운동 보충식 가이드 (타이밍별)
| 보충식 | 운동 중·전후 | 평소·간식 |
|--------|------------|----------|
| 포도당 캔디 | △ 빠른에너지(고강도 중) | ✕ 혈당스파이크·충치 |
| 게토레이(이온음료) | △ 땀 많은 운동시 전해질 | ✕ 설탕덩어리(당 과다) |
| 에너지젤리 | △ 장시간 운동 중 | ✕ 정제당 |
| 단백질바 | ○ 운동후 단백질(당 적은것) | △ 당 높으면 주의 |

### 타이밍 처방 (우리 증량 근거와 연결)
- ★운동 전: 탄수 보충(고구마·바나나) → 공복운동 근손실 방지
- 운동 중(1시간 이상 고강도): 빠른당(젤리·캔디·게토레이)으로 에너지 유지
- ★운동 직후: 빠른탄수 + 1시간내 단백질 → 근생성·증량 골든타임
- 평소: 위 제품들 간식 X (혈당스파이크·미각마비·충치)

### ★주의 (운동선수 부모 채널 핵심)
- 게토레이 한 병에 설탕 다량 → "운동 안 할 때 음료수처럼" 마시면 독
- 단백질바: 당 함량 확인 (당 6g 이하·단백질 높은 것)
- 어린 선수: 물+밥이 기본. 보충식은 고강도·장시간 운동 때만 보조
- ★"운동할 때만, 운동하니까" 먹는 것 (평소 간식 아님)

### ★우리 메시지 (타이밍의 과학)
- "나쁜 음식은 없다, 나쁜 타이밍이 있다" (운동 보충식 한정)
- 단, 평소엔 3대 백색음식(설탕) 그대로 적용
- ★엔진 처방: 운동 강도·시간 입력 → 보충식 필요여부·타이밍 안내

## 18. ★우유 vs 두유 (2026-06-23) — "우유 1~2잔" 처방의 완성

### ★유당불내증 = 핵심 분기점 (마른 아이 多)

① 포만감으로 식사량↓ (가장 큰 요인)

## 21. ★생활습관 4종 (스마트폰·좌식·배변·카페인) (2026-06-23)
═══════════════════════════════════════

### A. ★카페인 (콜라·에너지음료·커피) — 키에 직접 악영향
- ★카페인이 철분·칼슘 흡수 방해 → 청소년 성장·뼈 건강 악영향

- 소아청소년은 체중 적고 내약성 낮아 부작용 큼
  - 30kg 미만이 콜라 2캔 카페인만으로 구역·구토·불안·심계항진
- 고카페인 음료 = 100mL당 카페인 15mg 이상
- 청소년 고카페인 → 수면방해(서파수면↓ = 성장호르몬↓) + 분노·폭력행동

### B. ★스마트폰 — 직접 방해 아님, '움직임을 뺏어' 간접적 악영향
- ★중요: 스마트폰(화면·전자파)이 성장을 직접 막는 것 아님
- 문제는 스마트폰이 '행동을 바꿔서' 생기는 간접 영향:
  ① 오래 보느라 안 움직임 → 활동량↓ → 성장판 자극 기회↓ (가장 핵심)

  ③ 고개 숙이고 봄 → 거북목·굽은등 자세 → 숨은키 손실

- 숏폼(쇼츠·틱톡·릴스) 중독성 강 → 사용시간 늘어남
- 출처: 질병청 디지털 과의존
- ★엔진: 활동량 저하의 '원인'으로 활용 (직접 성장방해로 단정 금지)

### C. ★좌식(오래 앉기) — 활동량·장 동시 저하

- 활동량↓ → 성장판 물리자극↓

### D. ★배변·변비 → 흡수력 지표
- 변비 심할수록: 식사횟수 적고 아침결식↑, 외식↑, 채소·과일 선호↓

## 22. ★수면 시간대 + 취침 전 야식 (2026-06-23) — 활동축(수면)
═══════════════════════════════════════
> 기존 수면 근거(서파수면 16.4%)의 실전 정리. "언제 자나 + 자기전 먹나"

### A. ★성장호르몬 황금 시간대
- ★밤 10시 ~ 새벽 2시 = 성장호르몬 황금시간
- 잠든 후 1~2시간 뒤 깊은잠(서파수면) 단계서 성장호르몬 최대 분비
- → 밤 10시엔 이미 깊은잠 상태여야 = 늦어도 9~10시 취침
- "10시에 눕기"가 아니라 "10시엔 깊이 잠들어 있기"

### 권장 취침 시간 (역산)
| 대상 | 취침 | 기상 | 수면 |
|------|------|------|------|
| 초등(6~12) | 밤 9~9시반 | 7시 | 9~10시간 |
| 중·고생 | 밤 10~10시반 | 7시 | 8~9시간 |
- 핵심 3원칙: ①밤10시전 깊은잠 ②규칙적으로 ③취침전 스마트폰 차단

### B. ★취침 전 야식 — 양날의 검
[나쁜 경우 — 대부분]
- 자기 직전(30분내) 과식·기름진 음식 → 위장 일함 → 깊은잠 방해 → 성장호르몬↓
- 혈당 올림 → 인슐린 → 성장호르몬 억제 (설탕·탄산 근거와 동일)
- ★야식으로 과자·탄산·라면 = 최악 (혈당+칼슘손실+미각마비 다 들어감)

[도움되는 경우 — 잘 고르면]
- 자기 1~2시간 전 + 가벼운 단백질
- 따뜻한 우유 한 잔 (트립토판→수면도움 + 단백질·칼슘) ★추천
- 바나나·소량 견과
- → 마른 아이 증량에 +, 잠 안 방해

### ★핵심 원칙
1. 자기 직전(30분내) 과식 = 금지 (깊은잠 방해)
2. 먹는다면 자기 1~2시간 전, 가볍게
3. 야식으로 과자·탄산·라면 = 최악
4. 따뜻한 우유는 예외적 OK (단백질+수면)
- ★마른 아이라고 자기 직전 먹이면 역효과 → 저녁 충분히 + 1~2h전 우유

### ★엔진 활용
- 활동축(수면) 처방 + 저장력(증량) 교차
- 마른 아이 증량: "자기 직전" 아니라 "저녁 충분 + 1~2h전 우유"
- 출처: 수면-성장호르몬 생리, 혈당-인슐린-GH (우리 기존 근거 종합)
