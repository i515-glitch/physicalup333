# 333TEST 90문항 전체 목록 (대표님 작성 원본)

## 선택지(유료 정밀판): 전혀 아니다(0) / 아니다(1) / 그렇다(2) / 매우 그렇다(3)
## 정방향: 매우 그렇다=높은점수 / 역방향: 뒤집어 계산(3-원점수)

═══════════════════════════════════════
## 3축 구조
═══════════════════════════════════════
- 흡수력 (30문항, 역방향 10개=33%) : 먹는 것을 받아들이는 힘
- 연소력 (30문항, 역방향 9개=30%) : 에너지를 태우는 힘
- 저장력 (30문항, 역방향 9개=30%) : 살로 쌓는 힘
