# PHYSICAL UP 333 — 기술 명세서 (System Spec) v12.0
**안티그래비티 개발팀 전달용 · 로컬 데스크톱 및 웹 자동화 서버 구축 지시서**

---

## 0. 한 줄 요약
온라인 설문 + 인바디 정보 입력을 받아, 실서버 환경에서 BioCode를 계산하고 5단계의 임팩트 있는 맞춤형 모바일 웹 리포트 및 카톡 연동 알림 서비스를 생성하는 자동화 시스템.

---

## 1. 전체 데이터 흐름
```
[사용자 입력]
  ├─ 설문 응답 (수면·활동·위장소화 24문항)
  ├─ 아동 키 / 몸무게 / 생년월일시 / 부모 키
  └─ 인바디 데이터 입력 (또래 에이스 격차 분석용)
         ↓
[엔진 연산]  P333 엔진 v12.0 Strict Concern Mode
  입력 데이터 → 선천40·대사35·생활25 → 3축 점수 → 1~3단계 판정 → BioCode (3자리)
         ↓
[27패턴 매칭]  코드 → 기질 캐릭터명 + 출현율 + 특성 처방
         ↓
[피지컬 격차 분석]  또래 에이스 선수 평균 대비 신장·체중·근육 격차 계산
         ↓
[5단계 모바일 웹 리포트 생성]  report.html 에 주입 (TOC 단일화 및 100일 스케줄 포함)
         ↓
[알림톡 예약 전송]  Solapi API 연동 ➡️ 지정 시각 일괄 전송
```

## 2. 핵심 모듈 명세

### 2-1. 엔진 연산 (BioCode 계산 v12.0)
- **소스**: P333 엔진 .xlsx의 [가중치 매트릭스] 시트 로직을 코드로 이식
- **입력 7종**: 유전키(부모키)·음양오행(만세력)·사상의학체질·심성정 / 인바디·위장설문·수면활동설문
- **축 가중치**: 선천 40% · 대사 35% · 생활 25% (합 100)
- **3축 배분 매트릭스**:
  - 선천: 유전키30 + 음양오행25 + 사상의학25 + 심성정20
  - 대사: 인바디50 + 위장설문50 (설문 분모 45.0 상향 적용으로 고득점 인플레이션 방지)
  - 생활: 수면설문70 + 위장설문10 + 인바디20 (설문 분모 45.0 상향 적용)
- **단계 컷오프 (v12.0 엄격Concern 밸런스 패치 적용)**:
  - 선천(40점): 0~24.0 = 1 (Low) / 24.0~32.0 = 2 (Mid) / 32.0~40.0 = 3 (High)
  - 대사(35점): 0~19.0 = 1 (Low) / 19.0~27.5 = 2 (Mid) / 27.5~35.0 = 3 (High)
  - 생활(25점): 0~14.0 = 1 (Low) / 14.0~19.5 = 2 (Mid) / 19.5~25.0 = 3 (High)
- **BioCode**: 선천-대사-생활 단계를 이어붙인 3자리 (예: 1-2-1)
- **만세력**: 생년월일시→오행 변환은 별도 만세력_DB.xlsx(3289행) 참조.

### 2-2. 5단계 콤팩트 스토리 전개 (report.html)
학부모(엄마) 관점에서 직관적이고 몰입감 높은 5단계 단일 평면 목차로 구성:
1. **BioCode 종합 진단 결과**: 아동의 BioCode `2-1-2` 카드 & 3대 작동력 점수 선언.
2. **또래 에이스 대비 피지컬 격차 추적**: 에이스 대비 부족한 격차 시각화로 결핍 인지 유도.
3. **타고난 몸의 결 (유전 · 체질)**: 중간부모키(MPH) 및 타고난 대사 경향성 해설.
4. **생활 행동 3축 밸런스 (영양 · 수면 · 운동)**: 후천적 일상 습관 불균형 점검.
5. **100일 격차 극복 스케줄 및 처방전**: 주간 극복 타임라인 및 3대 집중 처방 제공 ➡️ 결제 및 쇼핑 전환 극대화.

### 2-3. 예약 및 지정 시간 일괄 카톡 발송 (Solapi ATA)
- **목적**: 기계적인 즉시 전송을 피해 신뢰도를 확보하고, 정성 어린 전문 분석 판독 시간 연출.
- **방식**: 매일 밤 지정된 스케줄 시점에 카카오 알림톡 API를 활용하여 예약 발송.

## 3. 무료 / 유료 구분 및 가치 퍼널
- **무료 테스트**: 5단계 콤팩트 스토리를 통해 아동의 핵심 BioCode 및 또래 대비 피지컬 결핍(1단계/2단계)을 명확히 고지하여 행동 유발.
- **유료 연동 및 쇼핑**: 100일 타임라인과 처방전 카드 바로 하단에 체질별 구글 엑셀 시트 연동 추천 영양 상품 구매 버튼을 다이내믹하게 노출하여 구매 유도.
