## ★ MLB 드래프트 컴바인 — 공식 체력측정 체계 (2026-06-23 추가)
═══════════════════════════════════════
> ★세계 최고 수준 야구 측정 체계. 우리 "운동능력 목표기준"의 표준.
> MLB는 묵직한 힘보다 "짧은 거리 폭발적 스피드(순발력) + 관절 회전 가동범위"를 중시.

### 1. 필드 성능·기초체력
| 항목 | 측정법 | 평균 | 상위10% |
|------|--------|------|---------|
| 30야드 대시 | 도루자세(옆선출발), 5야드마다 센서 | 3.77초 | 3.50~3.60초 |
| 제자리멀리뛰기(Broad Jump) | 고정자세 수평 하체파워 | 272cm(107인치) | 300~312cm(119~123인치) |
| 제자리높이뛰기(CMJ) | Force Plate 위 하체 폭발파워 | - | (지면반발력 측정) |
| 악력 | 손목·전완근 (배트스피드·회전) | - | - |

### 2. 가동성·부상위험 (MLB 최중시)
- **FMS**(Functional Movement Screen): 7동작 0~3점, 총 21점. **14점 미만=부상위험↑**
- **Y-밸런스 테스트**: 한발 서서 3방향, 동적균형 (투수 디딤발 안정성)
- **부위별 가동범위(ROM)**: 어깨(회전근개)·팔꿈치·고관절·흉추 회전각도

### 3. 야구특화 인지·신경반응
- **Man in the Box**: 4개 센서패드, 불빛/숫자 인지 후 터치 속도 (타격 반응속도)
- 반응시간(Reaction Time) = 현대야구 핵심 지표

### ★ 20-80 스카우팅 스케일 (= 우리 백분위 시스템의 원형!)
| 점수 | 의미 |
|------|------|
| 50 | MLB 전체 평균 |
| 60 | 평균 이상 우수 |
| 70+ | 리그 최상위권 (특급) |
| 30~40 | 빅리그 기준 미달 |
- ★우리 BioCode·비교군도 이 발상: 측정값→표준점수 환산→레벨 위치
- ★우리가 만들 "선수 또래 상위10%"가 MLB는 이미 70+ 스케일로 운영중

### ★엔진 활용
- 비교군(Ⅳ장) 측정항목 = 30야드·멀리뛰기·악력·FMS (야구 표준)
- 목표기준(Ⅵ장) = 컴바인 평균(50점)~상위(70점) 사다리
- 우리 백분위 표시 = 20-80 스케일 차용 (50=평균 기준)
- ★출처: MLB Draft Combine 공식 / 20-80 scouting scale

### 종목 적용 우선순위
- 야구: 컴바인 체계 그대로 (가장 정교)
- 타 종목: 해당 종목 컴바인/측정표준 추후 수집

## ★ MLB 컴바인 — 부위별 정밀 측정 메커니즘 (2026-06-23 추가)
═══════════════════════════════════════
> 현대야구 핵심관점: "지면 미는 힘(지면반발력) + 관절 회전 가동성" = 성능·부상방지

### 1. 하체 — 폭발력·지면반발력
- **CMJ(제자리높이뛰기)**: Force Plate 위, 지면 미는 힘(Newton) + 양발 균형(L/R) 실시간 측정
- **Broad Jump(멀리뛰기)**: 수평 추진력 (도루 첫보폭/마운드 박차기)
- ★타격·투구의 모든 힘은 하체에서 시작 → 위로 전달

### 2. 어깨·팔꿈치 — 투구 가동범위(ROM), 부상 최다부위
- 고니오미터(관절각도계)로 내회전/외회전 각도 정밀측정
- 던지는 팔: 외회전↑ 내회전↓ 경향
- ★GIRD 현상: 양팔 내회전 차이 20도↑ → UCL(팔꿈치인대) 파열위험 → 감점
- → 우리 보고서 "부상위험 평가" 항목 근거

### 3. 코어·골반 — 동적균형·회전력
- 고관절 가동성: 타격 시 골반 회전 → 배트 회전력 전달
- Y-밸런스: 한발 서서 3방향(전·후측·외측) 밀어낸 거리(cm)로 코어 안정성 수치화

### 4. 손목·전완근 — 악력
- 전자악력계 양손 측정
- ★MLB 컴바인 우수기준: **65~70kg 이상**
- 타격 컨택트 순간 힘 + 투구 릴리스 실밥 채는 힘

### ★엔진 활용 (보고서 측정·목표 항목 확정)
| 부위 | 측정 | 목표기준 |
|------|------|---------|
| 하체 | CMJ·멀리뛰기 | 멀리뛰기 평균272/상위300cm |
| 어깨 | 내외회전 ROM | 내회전차 20도 미만(GIRD 회피) |
| 코어 | Y밸런스 | 밀어낸 거리 cm |
| 악력 | 전자악력계 | 65~70kg+ (우수) |
- ★우리 비교군(Ⅳ장)·목표(Ⅵ장)에 부위별로 적용. 청소년은 연령보정 필요.

## ★ 일반 체력 기준 사다리 (선수 아래 구간 — 2026-06-23 추가)
═══════════════════════════════════════
> MLB(선수 끝판)부터 일반인까지 전 구간 사다리 완성용.
> ★주의: 아래는 성인(19~39세) 기준. 청소년은 연령보정 필요(질병청 학생건강체력 PAPS 별도).

### A. 국민체력100 — 일반 성인 남성 (19~39세)
| 체력요인 | 항목 | 우수(상위20%) | 평균(상위50%) |
|---------|------|--------------|--------------|
| 근력 | 악력(양손평균) | 50kg+ | 43~44kg |
| 근지구력 | 윗몸일으키기(1분) | 55회+ | 43~45회 |
| 심폐 | 셔틀런(20m왕복) | 60회+ | 45~48회 |
| 유연성 | 앉아윗몸앞으로굽히기 | 18cm+ | 10~11cm |
| 순발력 | 제자리멀리뛰기 | 240cm+ | 215~220cm |
(여성 우수: 악력31·윗몸43·셔틀35·유연22·멀리뛰기175)

### B. 전문/엘리트 수행능력 (상급자)
- 3대 운동(스쿼트·데드·벤치): 평균=벤치 체중1배·스쿼트 1.2배 / 엘리트=3대합 체중 4.5~5배
- 푸시업(1분): 남60·여45회 = 최상위
- 전완플랭크: 3분30초+ = 완벽한 코어
- VO2max: 일반35~40 / 선수·동호인 55~60ml/kg/min+

### C. 미 공군 체력테스트 (군 엘리트, 최대점수 기준 / 25세 미만 남)
| 항목 | 최대점수 기준 |
|------|--------------|
| 1.5마일 달리기 | ≤9:12 |
| 셔틀런(20m) | ≥100회 |
| 팔굽혀펴기(1분) | ≥67회 |
| 핸드릴리즈 푸시업(2분) | ≥40회 |
| 윗몸일으키기(1분) | ≥58회 |
| 크로스레그 리버스크런치(2분) | ≥49회 |
| 전완플랭크 | ≥3:35 |
(연령 올라갈수록 기준 완화 — 표 원문 보유)

### ★ 통합 사다리 (운동능력)
```
일반평균(국민체력100 50%) → 일반우수(20%) → 군엘리트(공군만점)
→ 선수또래상위10% → 대학 → KBO → NPB → MLB컴바인
```
- ★핵심: 5대 체력요인 "육각형 밸런스"가 진짜 '운동 잘하는 몸'
  (근력·근지구력·심폐·유연성·순발력 — 한쪽만 세면 안됨)
- 청소년 측정: 국민체력100 인증센터 무료 측정 가능 / 학교 PAPS

### ★엔진 활용
- 우리 레이더차트(6각형) = 이 "5대 체력 밸런스" 개념 그대로
- 현재값을 사다리 어디에 얹을지 → 일반~선수 전구간 비교 가능
- 셔틀런·악력·멀리뛰기·플랭크 = 공통 측정항목(일반~군~선수 다 씀) → 우리 기본 측정셋

## ★★ PAPS 학생건강체력평가 (청소년 기준 — 빈칸 해결!) 2026-06-23
═══════════════════════════════════════
> ★우리 청소년 고객의 핵심 기준. 성인 기준의 청소년 보정 문제를 한 방에 해결.
> 교육부 의무평가. 부모가 NEIS에서 데이터 바로 확인 가능 → 입력 쉬움!

### 측정 항목 (= 우리 측정셋과 거의 일치!)
- 심폐지구력: 왕복오래달리기(셔틀런/MSFT), 오래달리기-걷기, 스텝검사 중 선택
- 유연성: 앉아윗몸앞으로굽히기(좌전굴)
- 근력·근지구력: 팔굽혀펴기, 윗몸말아올리기, 악력
- 순발력: 50m달리기, 제자리멀리뛰기
- 체성분: BMI, 체지방률
→ 5개 체력요인 × 12종목 중 5개 선택 (요인별 1개씩)

### 5등급 체계 (= 우리 5단계와 정합)
| 등급 | 의미 |
|------|------|
| 1등급 | 매우우수 (상위권 기초체력) |
| 2등급 | 우수 |
| 3등급 | 보통 |
| 4등급 | 미흡 (보강 필요) |
| 5등급 | 매우미흡 (전문 처방·관리) |

### 중학생 기준 수치 (일부 확보)
- 악력: 남 35kg / 여 25kg (평균 기준)
- 셔틀런: 남 50~60회 / 여 40~50회 (3등급 기준)
- 제자리멀리뛰기: 고3남 3등급 221cm+ (성인평균 반영)
- ★연령·성별별 1~5등급 전체 커트라인표 = NEIS·교육부 PDF에서 확보 필요

### ★PAPS가 이미 하는 것 = 우리의 '선수 버전' 원형
- "맞춤형 운동 처방 제공 시스템" (단순 측정 아님)
- 개인별 결과표: 등급 + 항목별 점수 + 강점·약점 + 운동 계획
- → 우리는 이걸 "선수 목표 기준"으로 업그레이드 (일반 건강 → 선수 도달)

### ★데이터 입력 경로 (중요!)
- 부모가 NEIS(나이스) 학부모서비스에서 자녀 PAPS 결과 확인 가능
- → 우리 접수폼에 "PAPS 등급/기록" 입력란 → 부모가 NEIS 보고 입력
- 인바디 없어도 PAPS로 기초체력 파악 가능 (보조 입력)

### 통합 사다리 최종 (청소년~MLB)
```
PAPS 5등급 → PAPS 1등급 → 선수또래상위10% → 대학 → KBO → NPB → MLB
```

- ★반대로 이미 몸 완성돼 성장여지 없으면(Value Peak) 기피
- ★★결론: 스카우터도 "현재"가 아니라 "성장 잠재력"을 본다
  = 우리 슬로건 "타고났으나 더 클 수 있다" + 부모키 + 살찌우기가
    바로 프로 스카우팅의 핵심 관점!! (우리 엔진 정당성 입증)

### ★엔진·보고서 활용
- Ⅵ장 목표: 포지션 선택(투수/야수/포수) → 해당 합격선 표시
- 격차추적: 현재 구속/30m/팝타임 vs 합격선 → 처방
- ★우리 차별점 = "성장 잠재력"을 수치화 (부모키+벌크업 프레임)
  = 스카우터의 직관을 데이터로 = 우리가 프로보다 앞선 지점
- 출처: KBO 스카우팅 트렌드 (공식 재검증 권장)

## ★ 대학 야구 입시 기준 (사다리 대학 칸 완성!) 2026-06-23
═══════════════════════════════════════
> 고교→대학→KBO 사다리의 '대학' 칸. 프로와 달리 '실적+기록' 정량평가 60~80%.
> 출처: KBSA 지침 + 주요대학(고려·연세·한양·단국·동국) 요강

### 1. 서류 통과 커트라인 (지원 자격)
- 팀성적: 전국대회(황금사자기·청룡기·대통령배·봉황대기) 16강 또는 8강+
- 투수: 고교 3년 통산 20~30이닝+, ERA 5.00~6.00 이하
- 야수: 고교 3년 통산 50~60타석+, 통산타율 0.200~0.230+

### 2. 실기(피지컬/기술) 합격권
**투수** (완성도·제구 중시)
- 직구구속: 안정권 140km+ / 최소 135~137km
- 제구: BB/9 4.0개 이하
- 신장: 안정권 182cm+ / 최소 178cm+

**야수** (주력·송구 핵심)
- 100m: 안정권 11.4~11.7초 / 최소 12초 초반
- 홈→1루: 안정권 4.0~4.1초(우타) / 최소 4.3초
- 수비: 펑고 포구후 정확 송구

**포수**
- 팝타임: 안정권 1.95~2.05초 / 최소 2.15초

### 3. 평가 배점 (1000점 만점 예시)
- 실적 400 + 개인기록 300 + 실기/피지컬 200 + 학생부/면접 100
- ★스포츠 최저학력제: 내신 과도히 낮으면(7~8등급↓) 과락 가능
  → 운동만 아니라 학업도 챙겨야 (부모 코칭 포인트)

### ★사다리 최종 완성 (전구간)
```
리틀·중학(120~125km) → 고교(130~140) → ★대학(140) → KBO(143~148) → NPB(145) → MLB(147)
일반평균(KOSIS) → PAPS 5~1등급 → 선수 상위10% → 대학 → 프로
```
### ★엔진 활용
- Ⅵ장 목표: 진로 단계(대학/프로) 선택 → 해당 기준
- 대학은 "실적+기록+피지컬+학업" 종합 → 우리 로드맵에 학업 변수 추가
- 출처: KBSA·대학요강 (공식 재검증 권장)

## ★★ 종목별 국가대표 피지컬 프로필 (KISS/진천선수촌) 2026-06-24
═══════════════════════════════════════
> ★대표님 통찰: "국대 자료+인바디 → 역으로 목표 스케줄 수립"
> 출처: 문체부·한국스포츠과학원(KISS)·진천선수촌

### 4대 종목군별 신체 프로필
| 종목군 | 대표 지수 | 체지방률 | 핵심 능력 | 시각적 특징 |
|--------|----------|---------|----------|------------|
| ★야구/투척 | 윙스팬·골격근량 | 12~16% | 회전파워·지면반발 | 거대 프레임·긴 팔다리 |
| 축구/농구 | 30m·민첩성 | 7~10% | 심폐지구력·방향전환 | 슬림 탄탄 근육질 |
| 유도/레슬링 | 3대중량·무산소 | 9~13% | 코어·대인압박 근력 | 두꺼운 몸통·낮은 무게중심 |
| 육상장거리 | VO2max | 5~7% | 유산소 대사효율 | 극단적 마른 체형 |

### 1. 야구/투척 (★우리 핵심)
- ★윙스팬 = 키보다 3~5cm 이상 큰 선수 多 (회전반경↑ = 구속·비거리)
- 악력: 65~75kg+ (손목 스냅)
- 제자리멀리뛰기: 270~290cm+ (하체 플라이오 파워)
- 측정지수: FFMI(제지방질량지수)·골격근량·악력·멀리뛰기

### 2. 축구/농구 (스피드·민첩)
- 체지방률 7~10% 이하 (데드웨이트 없음)
- 수직점프 70~80cm+ (농구가드·배구 85~90cm+)
- 10m 1.6초대 / 30m 3.7~3.8초 돌파

### 3. 유도/레슬링 (코어·내구)
- 윙스팬 키와 비슷하거나 짧음(지레 작용 회피)
- 벤치프레스 체중 1.5~1.8배+ / 데드·스쿼트 2.2~2.5배+
- 윈게이트 피크파워 13~15W/kg+

### 4. 육상장거리 (지구력)
- ★VO2max 70~80 ml/kg/min+ (일반 성인 40 → 2배, '인간 기관차')
- 체지방률 5~7%

### ★★ 엔진 핵심 활용 (대표님 통찰 = 역산 목표 스케줄)
```
[현재] 인바디 측정 (체지방률·골격근량·윙스팬·악력 등)
   ↓ 비교
[목표] 종목 국대 프로필 (야구=체지방12~16%·윙스팬+3~5cm·악력65+)
   ↓ 격차 산출
[역산] 목표까지 필요한 변화량 → 나이별 보간 → 단계별 스케줄
   예: "현재 체지방 22%·악력 40 → 야구목표 16%·65 →
        3년 로드맵: 중3 체지방 19%, 고1 17%, 고3 16% / 악력 +8kg/년"
```
- ★FFMI·골격근량·윙스팬 = 인바디로 측정 가능 → 바로 목표 대비 격차
- ★우리 BioCode(흡수·연소·저장)와 연동:
  체지방 높음=저장3 / 마른선수=저장1 → 종목 목표 체지방으로 조정
- ★종목별 베스트코드 정당화: 야구332(파워·12~16%) / 축구221(슬림7~10%)
  / 마라톤211(극마름5~7%) = 국대 체지방률과 정확히 일치!
- 출처: KISS·진천선수촌 (공식 재검증 권장)
