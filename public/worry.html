02Mechanism
우리 아이는 왜 작은가
동일한 식이를 섭취해도 결과가 다른 이유는 영양 처리 메커니즘의 개인차에 있습니다. 흡수·연소·축적 3단계 효율이 BIO CODE를 결정합니다.
01
흡수 ABSORPTION소화·장내 환경
장 점막 손상 또는 소화효소 부족 시 영양 흡수율이 최대 20–30% 저하됩니다. 고단백 식이를 해도 실제 근육 합성에 도달하는 영양소는 절반 이하가 됩니다.
02
연소 METABOLISM기초대사·에너지 전환
미토콘드리아의 ATP 전환 효율에 따라 결정됩니다. 연소 과잉형은 소비형(BURNER), 연소 저하형은 저장형(STORER)으로 분류됩니다.
03
축적 STORAGE근·골격 합성
잉여 에너지가 근육·골격으로 가는지 체지방으로 가는지를 결정합니다. 성장호르몬은 심야 깊은 수면 중 분비되며, 아연·마그네슘·비타민D 결핍 시 합성이 억제됩니다.

03Diagnosis · Free
데이터로 측정한다
표준화된 18문항 설문으로 흡수·연소·축적 3축 점수를 산출하고, 27가지 BIO CODE 중 대상의 코드를 결정합니다.
1
표준 설문 측정
90문항 검증 풀에서 축별 6문항씩 무작위 추출. 정·역방향 문항을 30% 혼합하여 응답 신뢰도를 확보합니다.
2
3축 점수 산출
각 축 0–18점을 1–3단계로 정규화합니다. (Absorption · Metabolism · Storage)
3
BIO CODE 결정
3×3×3 = 27개 코드 좌표 중 대상의 코드를 매핑합니다.
▎27-CODE 3D MATRIX — 코드를 입력해 좌표를 확인하세요
BURNER BALANCED STORER SELECTED
3자리 코드를 입력하면
3D 좌표와 특성이 표기됩니다.

예: 111 · 222 · 333

운동 처방근력 대 유산소 최적 비율
