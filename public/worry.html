Physical is Data
우리 아이의 성장과 퍼포먼스, 감이 아니라 데이터로 분석합니다. BIO CODE로 시작하는 과학적 관리.
STEP 01
우리 아이가 얼마나 작을까?
나이별 키·몸무게를 또래 평균, 상위 10%, 우수 선수와 비교해보세요. 우리 아이의 현재 위치를 데이터로 확인합니다.

※ 교육부 학생건강체력평가(PAPS) · 대한성장의학회 성장곡선 · 대한체육회 엘리트선수 기준

STEP 02
우리 아이는 왜 작을까?
아이마다 음식을 소화·흡수하는 정도, 에너지를 쓰는 양, 저장하는 양이 다릅니다. 이 3가지가 BIO CODE를 결정합니다.
1
흡수력 — 얼마나 잘 흡수하는가
장 환경이 나쁘면 아무리 먹어도 흡수율이 20~30%까지 떨어집니다. 좋은 음식도 절반 이상 그냥 빠져나갑니다.
2
연소력 — 얼마나 빨리 태우는가
연소가 너무 빠르면 살이 안 찌고(소비형), 너무 느리면 지방으로 쌓입니다(저장형).
3
축적력 — 얼마나 잘 저장하는가
남은 에너지가 근육·뼈로 가는지, 지방으로 가는지를 결정합니다. 성장호르몬이 핵심입니다.
BIO CODE분포특징관리 방향
소비형
BURNER		47%잘 안 찌는 체질흡수 강화·근력 운동
균형형
BALANCED		31%이상적 균형잔근육 강화 유지
저장형
STORER		22%잘 찌는 체질대사 활성·유산소

BIO CODE를 모르면, 아무리 먹여도 효과 없는
"밑 빠진 독에 물 붓기"가 됩니다.

STEP 03 · 무료
우리는 DATA로 분석한다
부모와 아이의 간단한 설문으로 우리 아이의 BIO CODE를 정확하게 측정합니다. 무료입니다.
1
333TEST 설문 (18문항)
90개 문항 풀에서 흡수·연소·축적 각 6개씩 랜덤 추출. 약 3~5분이면 충분합니다.
2
3축 점수 산출
흡수력·연소력·축적력을 각각 1~3점으로 계산합니다.
3
BIO CODE 결정
3×3×3 = 27가지 코드 중 우리 아이의 코드가 나옵니다.
📦 27가지 BIO CODE 큐브 — 코드를 입력해보세요

코드 3자리를 입력하면
큐브에서 위치와 특성이
